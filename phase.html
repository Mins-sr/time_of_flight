位相を読む

超短時間を、波形のズレ角として読む

72.0 deg
このページで学ぶこと

位相とは、1周期を360度で表したときの戻り光のズレです。距離D → 往復時間τ → 位相差φ → 復元距離Dの順に見ます。

waveform

送信光と戻り光

図の読み方: 青=送信、赤=戻り、横軸=時間、1周期=360度です。スライダーを動かすと、戻り光のズレ角が変わります。

変調 f は、光を1秒に何回点滅させるかです。fを上げると位相差は大きく見えますが、一意範囲は短くなります。

  1. 距離Dを決める
  2. 往復時間τ = 2D / c を出す
  3. τを1周期の角度φに変える
  4. φから距離Dへ戻す
numbers

計算値

周期 T = 1 / f 100.000 ns
往復時間 τ = 2D / c 20.000 ns
復元距離 3.000 m
一意範囲 15.00 m

D = 72.0 / 360 x 15.00 = 3.000 m

復元距離は、測った位相差φから計算で戻した距離です。一意範囲は、位相だけで区別できる最大距離です。10MHzでは15mまでが折り返さずに読めます。
