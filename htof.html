hTOFの考え方

このページで学ぶこと

hTOFはhybrid Time-of-Flightの略です。だいたい何m台かを表す粗い距離帯 gate と、その中の細かい割合 fine を組み合わせて距離を出します。

gate ruler

粗いゲート + N3比率

図の読み方: gate番号は0mから始まります。gate 3 は3m以上4m未満の区間です。青い線が、その区間内でどこまで進んだかを表します。

D = (3 + 0.25) x 1.00 = 3.25 m

TP = 6.667ns は、1m幅のゲートを作るための時間幅です。

charge split

N2 と N3

N2 ゲート前半の信号量
N3 ゲート後半の信号量

fine = N3 / (N2 + N3)

D = (gate + fine) x 1m

fine は「そのgate内の細かい割合」です。N2とN3は戻り光から得た信号量で、N3の割合が大きいほど物体はそのgateの後ろ寄りにあります。
