このページで学ぶこと

iToFは、光を出して戻り光の遅れから距離を求める方式です。ここでは「位相 → 式と表 → 画素の世界 → hTOF」の順に、距離が画像になるまでを追います。

01

位相

周期的な光のズレを、時間ではなく角度として読む。

D = φ / 360 x c / 2f D=距離、φ=ズレ角、c=光速、f=点滅の速さ 02

式と表

1mから8mまでの往復時間、位相差、復元距離を確認する。

1m -> 6.667ns -> 24deg 03

世界

640 x 480 の各画素が、それぞれ1つの距離値を持つ。

pixel(x, y) -> D pixel(x, y)=画面上の1画素、D=その画素の距離 04

hTOF

粗い距離帯と、その中の細かい位置を足し合わせる。

(gate + ratio) x width gate=何m台か、ratio=その中の割合、width=1m幅
snapshot

8個の積み木

図の読み方: 明るい四角が20cm角の積み木です。色が近い/遠いを表し、各四角は1m, 2m, 3m...8mの距離を持ちます。

flow

測距の流れ

  1. 光を出す10MHzなら100ns周期
  2. 反射して戻る1mなら往復2m
  3. 位相を読む1mなら24deg
  4. 距離へ戻すD = φ / 360 x 15m
